# Atlas Task Navigator

Atlas is a task management application designed to help you organize your daily life. Whether you need to track simple To-Dos, manage Deadlines, or schedule Events, Atlas keeps everything in one place with a clean, intuitive GUI.



## Key Features
* **Task Management:** Add, delete, and mark tasks as complete or incomplete.
* **Smart Search:** Easily find tasks using keywords.
* **Persistent Storage:** Your data is automatically saved to a local text file and reloaded upon startup.
* **Sorting:** Keep your schedule organized by sorting tasks alphabetically.

## How to Use Atlas

### Prerequisites
* **Java 17 or higher** must be installed on your machine.

### Running the Application
1. Download the latest `atlas-all.jar` from the [Releases](https://github.com/riazhaq/ip/releases) page.
2. Place `atlas-all.jar` in an empty folder on your computer.
3. Open your terminal or command prompt in that folder.
4. Run the following command:
   ```bash
   java -jar atlas-all.jar
